# 🚀 MV Software Frontend

A modern, high-performance frontend built using **Next.js 14**, designed to deliver a sleek, responsive, and interactive user experience.

---

## 📌 Overview

This project represents the frontend architecture for **MV Software**, focusing on:

- Clean UI/UX
- High performance
- Scalable component structure
- Smooth animations and transitions

It is built using modern frontend technologies and best practices to ensure maintainability and speed.

---

## ✨ Features

- ⚡ Fast rendering with Next.js (SSR & SSG)
- 🎨 Responsive and modern UI design
- 🎬 Smooth animations (Framer Motion + GSAP)
- 🧩 Reusable component architecture
- 🧠 Clean and scalable codebase
- 🔥 Interactive UI elements (Radix UI)
- 🌐 Supabase integration ready
- 📱 Mobile-first design

---

## 🛠 Tech Stack

### Frontend
- Next.js 14
- React 18
- Tailwind CSS

### UI & Animation
- Framer Motion
- GSAP
- Radix UI
- Swiper
- Lenis (smooth scrolling)
- Atropos

### Backend / Services
- Supabase (JS + SSR)

### Utilities
- Lucide React (icons)
- Class Variance Authority
- Tailwind Merge

## ⚙️ Getting Started

### 🔧 Prerequisites

- Node.js (v18+ recommended)
- npm / yarn / pnpm / bun

---

### 📥 Installation

```bash
git clone https://github.com/Mayurv153/mvsoftware-frontend.git
cd mvsoftware-frontend
npm install

▶️ Run Development Server
npm run dev

🏗 Build for Production
npm run build
npm run start
🧹 Lint Code
npm run lint
